# Movie Find

A web application that searches movies by a title query and returns information on the movie.

[Check it out here!](https://slyty7.github.io/MovieFind/)

## Built With

  * HTML
  * CSS
  * JavaScript
  * jQuery
  * Bootstrap
  * Bootswatch(Bootstrap Theme)
  * Axios
